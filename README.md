<h1 align='center'>
    Display location of .git's on device
</h1>

<h4 align='center'>
    Losing clones? foolgit will find it for you.
</h4>

## Display

```
foolgit list
```

## Paths

```
<location>/<group>/<repo_name>/.git/
```
